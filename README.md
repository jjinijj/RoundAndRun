# RoundAndRun

Unity 기반 엔들리스 러너 게임. 자동으로 전진하는 플레이어가 장애물을 점프/슬라이드로 피하고 아이템을 수집합니다.

> **현재 개발 중인 프로토타입입니다.**

**플레이 링크**: https://round-and-run.vercel.app/

## 개발 예정

- **맵 패턴 다양화** — 현재 고정 패턴 반복에서 다양한 패턴으로 개선
- **난이도 상승** — 오래 달릴수록 맵 이동 속도 증가
- **아이템 다양화** — 아이템 종류 및 배치 패턴 확대
- **UI 개선**

## 조작법

| 액션 | 키보드 | 모바일 |
|------|--------|--------|
| 점프 | `Space` | Jump 버튼 |
| 슬라이드 | `↓` | Slide 버튼 |
| 게임 시작 | `S` | - |
